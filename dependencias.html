Dependencias de Spring Boot

Este es un resumen de las dependencias más comunes que puedes añadir a tu proyecto de Spring Boot. Para más detalles y configuración personalizada, visita Spring Initializr.

Núcleo y Web
  • spring-boot-starter: Proporciona el núcleo de Spring Boot, incluida la autoconfiguración y el soporte para archivos YAML.
  • spring-boot-starter-web: Facilita el desarrollo de aplicaciones web.
  • spring-boot-starter-web-services: Pensado para la creación de servicios web SOAP.
Acceso a Datos
  • spring-boot-starter-data-jpa: Proporciona soporte para el mapeo objeto-relacional con Spring Data JPA y Hibernate.
  • spring-boot-starter-data-mongodb: Pensado para interactuar con bases de datos MongoDB.
  • spring-boot-starter-data-redis: Ofrece soporte para trabajar con bases de datos Redis.
  • spring-boot-starter-data-cassandra: Permite la integración con bases de datos Cassandra.
  • spring-boot-starter-data-elasticsearch: Facilita la interacción con servidores Elasticsearch.
Seguridad
  • spring-boot-starter-security: Integra Spring Security para proveer características de autenticación y autorización.
Plantillas y Vistas
  • spring-boot-starter-thymeleaf: Permite usar Thymeleaf como motor de plantillas.
  • spring-boot-starter-freemarker: Integra el motor de plantillas FreeMarker.
Herramientas para Desarrolladores
  • spring-boot-devtools: Facilita el desarrollo proporcionando recarga automática de clases y otros recursos.
Streaming de Datos
  • spring-boot-starter-stream-kafka: Para aplicaciones que usan Apache Kafka para el streaming de datos.
  • spring-boot-starter-stream-rabbit: Para aplicaciones que usan RabbitMQ para el streaming de datos.
Cliente HTTP
  • spring-boot-starter-webclient: Facilita el uso de WebClient para realizar peticiones HTTP de forma reactiva.
Datos Reactivos
  • spring-boot-starter-data-mongodb-reactive: Ofrece soporte para trabajar con MongoDB de manera reactiva.
  • spring-boot-starter-data-cassandra-reactive: Permite usar bases de datos Cassandra de manera reactiva.
Web Reactiva
  • spring-boot-starter-webflux: Para construir aplicaciones web reactivas.
Mensajería y Eventos
  • spring-boot-starter-amqp: Proporciona el soporte necesario para la mensajería AMQP, como RabbitMQ.
  • spring-boot-starter-websocket: Ofrece soporte para trabajar con WebSockets.
Actuadores y Monitorización
  • spring-boot-starter-actuator: Proporciona endpoints de actuación y métricas para monitorizar la aplicación.
Pruebas
  • spring-boot-starter-test: Agrupa diversas bibliotecas útiles para pruebas, como JUnit, Mockito y Hamcrest.
Correo Electrónico
  • spring-boot-starter-mail: Facilita el envío de correos electrónicos a través del servicio de correo JavaMail.
Servidores Web Alternativos
  • spring-boot-starter-jetty: Utiliza Jetty como servidor web.
  • spring-boot-starter-undertow: Utiliza Undertow como el servidor web.
